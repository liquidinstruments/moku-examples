Logic Analyzer
===========================

The Logic Analyzer / Pattern Generator can measure or generate up to 16
channels of digital signal. The output digital signal can be programmed to
Low, High, or a custom pattern

#### Available states for a pin are:

| State | Description             |
| ----- |:---------------------- |
| I     | Input                   |
| O     | Output                  |
| H     | High, pin is set to 1   |
| L     | Low, pin is set to 0    |
| X     | Off, Pin is off         |
